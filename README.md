# README

Materials for week 01 of Intro to Data at the Albany College of Pharmacy and Health Science.

- [Lecture](./lecture.qmd)
  - Introduction to the course, goals, and objectives.
  - Introduction to data and data science.
  - Instructor uses Excel to discuss row-level as well as class averages, comparisons between groups, etc.
- [Lab](./lab.qmd)
  - Gives students the opportunity to learn the RStudio interface and start thinking about data manipulation and graphing using R.
- The data contained in [Class Grades](./data/class_grades.xlsx) is synthetic data. 
  - The data was created by the script, [class_grades](/.data/class_grades.R).
  - Synthetic students are not covered by [FERPA](https://studentprivacy.ed.gov/ferpa).
